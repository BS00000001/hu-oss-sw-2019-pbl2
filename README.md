# hu-oss-sw-2019-pbl2
## server
1. clone
	```bash
	$ git clone -b server https://github.com/BS00000001/hu-oss-sw-2019-pbl2
	```
2. install packages
	```bash
	$ npm i
	```
3. mongodb setting
4. modify '.env' file
5. start mongodb
	```bash
	$ mongod (--dbpath=..)
	```
6. start server

	```bash
	$ npm start
	```
